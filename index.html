Current & Recent Projects

  • Flow Score Analyzer for Rap Lyrics – Python + NLP tool that quantifies rhyme density, phonetic overlap, multisyllabic structures, and flow segments. Web app lets users paste lyrics, process, and view interactive visual breakdowns; long‑term vision includes premium insights & TouchDesigner visualizations.
  • Lyrics Upload & Processing Web App – Flask backend saves user‑named lyric files, queues analysis jobs, returns HTML results with progress/loading UI.
  • Basketball Data Scraper & Dashboard – Automated Basketball Reference scraping (team rosters by season), standardized historical team abbreviations (e.g., NOP ↔ NOK/NOH), and an interactive dashboard for exploring CSV outputs.
  • College Football Data Explorer – Pulls structured JSON from SportsDB API; foundation for comparative team/event visualizations.
  • Weather & Aviation Heat Study – Using 20+ years of NCEI climate data and airport landing counts to test predictive heat models with scikit‑learn (and alternatives for long time‑series).
  • Company Bubble Viz – Canvas/SVG experiment that draws company "balls" sized by employee count; born from exploring quick visual storytelling in the browser.
  • Solana Token Experiment – Early‑stage exploration of launching a small crypto token on Solana; focusing on tooling, wallet flows, and lightweight utility concepts.
  • Responsive Physics UI Experiments – matter.js and custom JavaScript motion tests to make interface elements behave with playful, physics‑based interactions.
